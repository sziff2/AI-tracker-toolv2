SCORING RUBRIC (apply to each agent):

Specificity (0-0.25):
  0.25 = cites specific numbers, names metrics, gives precise ranges
  0.15 = some specifics but also vague generalisations
  0.05 = mostly vague ("could improve", "might be challenging")

Evidence basis (0-0.25):
  0.25 = every claim traceable to extracted data or management quotes
  0.15 = most claims supported, some assertions
  0.05 = largely opinion-based with few data references

Thesis linkage (0-0.25):
  0.25 = directly addresses thesis pillars, updates assumptions
  0.15 = references thesis but loosely
  0.05 = generic analysis disconnected from investment thesis

Actionability (0-0.25):
  0.25 = clear "so what" — specific monitoring items, triggers, timelines
  0.15 = some actionable items mixed with generic observations
  0.05 = descriptive only, no clear next steps

CONSISTENCY CHECK:
- If Context Contract says "higher_for_longer" rates, flag any agent
  assuming rate cuts
- If Context Contract says USD "strengthening", flag any agent assuming
  USD weakness
- If Context Contract says credit "tightening", flag any agent assuming
  easy refinancing

CITATION CHECK (Tier 4.4 — pilot on bear_case, rolling out later):
Agents that emit a `sources` array must have their claims traceable
to the cited data. The CITATION RESOLUTION REPORTS block above gives
